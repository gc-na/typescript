# TypeScript 中的 "for" 循環：使用指南

## 概述
在 TypeScript 中，`for` 循環是一種基本的控制結構，允許開發者重複執行一段代碼，直到滿足特定的條件。它是處理數組和集合數據的理想選擇，並且在性能優化方面表現良好。

## 文檔
`for` 循環的主要目的是提供一種簡單的方法來重複執行一段代碼。基本的語法結構如下：

```typescript
for (初始語句; 條件語句; 更新語句) {
    // 代碼塊
}
```

- **初始語句**：在循環開始前執行一次，通常用來初始化計數器。
- **條件語句**：每次執行循環時檢查，當條件為真時，執行循環內的代碼塊。
- **更新語句**：在每次循環結束時執行，通常用來更新計數器。

### 使用方法
`for` 循環可用於遍歷數組、對象或任何可迭代的數據結構。以下是一些用法示例：

```typescript
// 遍歷數組
const numbers: number[] = [1, 2, 3, 4, 5];
for (let i = 0; i < numbers.length; i++) {
    console.log(numbers[i]);
}
```

## 範例
以下是一些 `for` 循環的基本使用範例：

1. **簡單計數循環**
    ```typescript
    for (let i = 0; i < 5; i++) {
        console.log(i); // 0, 1, 2, 3, 4
    }
    ```

2. **遍歷數組**
    ```typescript
    const fruits: string[] = ['蘋果', '香蕉', '橙'];
    for (let i = 0; i < fruits.length; i++) {
        console.log(fruits[i]); // 蘋果, 香蕉, 橙
    }
    ```

3. **倒序循環**
    ```typescript
    for (let i = 5; i > 0; i--) {
        console.log(i); // 5, 4, 3, 2, 1
    }
    ```

## 解釋
在使用 `for` 循環時，有些常見的陷阱和注意事項包括：

- **無限循環**：如果條件始終為真，則循環將無法停止，這會導致程序掛起。
- **範圍問題**：確保計數器的範圍正確，避免越界錯誤（例如，訪問數組中不存在的索引）。
- **變量作用域**：如果在 `for` 循環中使用 `let` 來聲明計數器，則計數器的作用域僅限於循環內部，這有助於避免意外修改。

## 一句總結
TypeScript 的 `for` 循環是一種強大且靈活的結構，用於執行重複操作並遍歷數據集合。
