# الوحدات في TypeScript: كل ما تحتاج معرفته

## ملخص
تعتبر الوحدات في TypeScript وسيلة لتنظيم الكود وإعادة استخدامه، مما يسهل تطوير تطبيقات كبيرة ومعقدة.

## الوثائق
تُستخدم الوحدات في TypeScript لتعريف نطاقات مختلفة من الكود. يمكن أن تحتوي الوحدات على متغيرات، وظائف، أو واجهات، مما يسمح بتقسيم البرنامج إلى أجزاء أصغر وأكثر قابلية للإدارة. يتم تعريف الوحدة باستخدام الكلمة المفتاحية `export`، مما يجعل الكود داخل الوحدة متاحًا للاستخدام في وحدات أخرى. يمكن استيراد الوحدات باستخدام الكلمة المفتاحية `import`.

### الغرض
- تنظيم الكود: تساعد الوحدات على تقسيم الكود إلى أجزاء منطقية.
- إعادة الاستخدام: يمكن استخدام الوحدات في عدة أماكن داخل التطبيق أو حتى عبر تطبيقات مختلفة.
- تحسين القراءة والصيانة: يسهل فهم الكود عند تقسيمه إلى وحدات صغيرة.

### الاستخدام
لتعريف وحدة في TypeScript، يتم استخدام الكلمة المفتاحية `export`، بينما يتم استخدام `import` لاستيراد الوحدات في ملفات أخرى.

```typescript
// تعريف وحدة في ملف math.ts
export function add(a: number, b: number): number {
    return a + b;
}
```

```typescript
// استيراد الوحدة في ملف آخر
import { add } from './math';

const result = add(5, 10);
console.log(result); // 15
```

## الأمثلة
### مثال 1: تعريف واستيراد وحدة بسيطة
```typescript
// في ملف greetings.ts
export function greet(name: string): string {
    return `Hello, ${name}!`;
}

// في ملف main.ts
import { greet } from './greetings';

console.log(greet('Alice')); // Hello, Alice!
```

### مثال 2: استيراد عدة وظائف من وحدة
```typescript
// في ملف mathOperations.ts
export function multiply(a: number, b: number): number {
    return a * b;
}

export function divide(a: number, b: number): number {
    return a / b;
}

// في ملف main.ts
import { multiply, divide } from './mathOperations';

console.log(multiply(6, 7)); // 42
console.log(divide(42, 6)); // 7
```

## الشرح
- **تعاريف متكررة**: تأكد من عدم استخدام نفس الاسم لوظائف مختلفة في وحدات متعددة، حيث يمكن أن يؤدي ذلك إلى تداخل الأسماء.
- **تصدير افتراضي**: يمكنك استخدام `export default` لتصدير وحدة واحدة كافتراضية، مما يسهل استيرادها.
- **الوحدات الدائرية**: تجنب الوحدات الدائرية، حيث يمكن أن يؤدي ذلك إلى صعوبات في التصدير والاستيراد.

## ملخص جملة واحدة
تساعد الوحدات في TypeScript على تنظيم الكود وإعادة استخدامه بسهولة من خلال تعريف وتصدير واستيراد الوظائف والبيانات.
