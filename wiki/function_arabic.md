# الدوال في TypeScript: دليل شامل لفهم واستخدام الدوال

## ملخص
الدوال في TypeScript هي وحدات برمجية مستقلة تُستخدم لتنفيذ مجموعة من التعليمات، مما يساعد في تنظيم الكود وإعادة استخدامه بفعالية.

## الوثائق
### الغرض
الدوال في TypeScript تُستخدم لتجميع التعليمات البرمجية التي يمكن استدعاؤها في أي مكان في البرنامج. يمكن أن تأخذ الدوال معلمات (Parameters) وتعيد قيمة (Return Value) أو لا تعيد أي شيء.

### الاستخدام
تُعرف الدالة باستخدام الكلمة المفتاحية `function`، تليها اسم الدالة، ثم قائمة المعلمات (إذا وجدت) ونوع القيمة المعادة. 

#### بناء الجملة:
```typescript
function اسم_الدالة(معامل1: نوع, معامل2: نوع): نوع_القيمة_المعادة {
    // تعليمات
    return قيمة;
}
```

### التفاصيل
- **اسم الدالة**: يجب أن يكون فريدًا ولا يحتوي على مسافات.
- **المعلمات**: يمكنك تحديد أنواع المعلمات، مما يوفر أمانًا أكبر عند استخدام الدالة.
- **نوع القيمة المعادة**: يُعتبر تحديد نوع القيمة المعادة مهمًا لضمان أن الدالة تُرجع نوع البيانات المتوقع.

## أمثلة
### مثال 1: دالة بسيطة
```typescript
function جمع(a: number, b: number): number {
    return a + b;
}

console.log(جمع(5, 10)); // الناتج: 15
```

### مثال 2: دالة بدون قيمة معادة
```typescript
function مرحبا(name: string): void {
    console.log(`مرحبا، ${name}!`);
}

مرحبا("أحمد"); // الناتج: مرحبا، أحمد!
```

## الشرح
### العقبات الشائعة
- **عدم تحديد نوع القيمة المعادة**: إذا لم يتم تحديد نوع القيمة المعادة، يمكن أن يؤدي ذلك إلى أخطاء في وقت التشغيل.
- **تجاوز عدد المعلمات**: عند استدعاء دالة بعدد غير صحيح من المعلمات، سيؤدي ذلك إلى سلوك غير متوقع.
- **المعاملات الافتراضية**: يمكن أن تكون المعلمات افتراضية، مما يتيح لك استدعاء الدالة بعدد أقل من المعلمات.

### ملاحظات إضافية
- يمكن استخدام الدوال كقيم، مما يعني أنه يمكنك تمريرها كمعلمات إلى دوال أخرى أو تخزينها في متغيرات.
- تدعم TypeScript أيضًا الدوال المجهولة (Anonymous Functions) والدوال السهمية (Arrow Functions)، مما يوفر مزيدًا من المرونة.

## ملخص بجملة واحدة
الدوال في TypeScript هي وسيلة قوية لتنظيم الكود وإعادة استخدامه، مما يسهل تطوير التطبيقات المعقدة.
