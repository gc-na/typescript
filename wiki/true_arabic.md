# القيمة "true" في TypeScript: الفهم والاستخدام

## ملخص
تعتبر القيمة "true" واحدة من القيم المنطقية الأساسية في TypeScript، المستخدمة بشكل واسع في التحكم في تدفق البرنامج والشروط.

## الوثائق
تُستخدم القيمة "true" في TypeScript كقيمة منطقية تشير إلى الحقيقة. TypeScript، كامتداد لـ JavaScript، يعتمد على القيم المنطقية مثل "true" و"false" لتحديد حالات معينة في الشيفرة. تُعتبر "true" جزءًا من نوع البيانات `boolean`، ويمكن استخدامها في الشروط، الحلقات، والوظائف.

### الغرض
الغرض من استخدام "true" هو تمثيل حالة إيجابية أو تأكيد شيء ما. يمكن دمجها في الشيفرات البرمجية لتوجيه تدفق البرنامج أو اتخاذ قرارات بناءً على تقييم الشروط.

### الاستخدام
يمكن استخدام "true" في العديد من السياقات، بما في ذلك:
- الشروط (if statements)
- الحلقات (loops)
- المتغيرات (variables)

### التفاصيل
يمكن استخدام "true" بشكل مباشر أو كمقابل لقيم أخرى. في TypeScript، يمكن استخدام "true" في الشروط كما يلي:

```typescript
let isActive: boolean = true;

if (isActive) {
    console.log("الحالة نشطة");
}
```

## أمثلة

### مثال 1: استخدام "true" في شرط
```typescript
let isLoggedIn: boolean = true;

if (isLoggedIn) {
    console.log("مرحبًا بك في حسابك!");
} else {
    console.log("يرجى تسجيل الدخول.");
}
```

### مثال 2: استخدام "true" في حلقة
```typescript
let continueLoop: boolean = true;
let count: number = 0;

while (continueLoop) {
    count++;
    if (count > 5) {
        continueLoop = false;
    }
}

console.log("عدد التكرارات: " + count);
```

## الشرح
على الرغم من بساطة استخدام "true"، هناك بعض النقاط التي يجب الانتباه إليها:
- **التأكد من النوع**: تأكد من أن المتغيرات التي تُستخدم مع "true" يتم تعريفها كنوع `boolean` لتجنب الأخطاء.
- **القيم المنطقية الأخرى**: "false" هي القيمة المعاكسة لـ "true" ويمكن استخدامها بشكل مكمل في الشروط.

## ملخص جملة واحدة
تُستخدم القيمة "true" في TypeScript للإشارة إلى حالة منطقية إيجابية، مما يسهل التحكم في تدفق البرنامج.
