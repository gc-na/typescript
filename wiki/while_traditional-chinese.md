# TypeScript 的 while 迴圈：使用方法與範例

## 概述
`while` 迴圈是 TypeScript 中用於重複執行一段程式碼的控制結構，直到指定的條件為假。這使得它成為處理重複性任務的強大工具。

## 文件說明
`while` 迴圈的基本語法如下：

```typescript
while (condition) {
    // 執行的程式碼
}
```

### 目的
`while` 迴圈允許開發者在不確定執行次數的情況下重複執行程式碼區塊，直到條件不再滿足。

### 使用方法
1. **初始化變數**：在迴圈外部定義並初始化用於條件判斷的變數。
2. **條件判斷**：`while` 後的條件會在每次迴圈開始前進行評估。
3. **迴圈內容**：若條件為真，則執行迴圈內的程式碼。
4. **更新條件**：在迴圈內部必須更新條件變數，以避免無限迴圈。

## 範例
### 基本範例
以下是一個簡單的 `while` 迴圈範例，用於計算從 1 加到 5 的總和：

```typescript
let sum = 0;
let i = 1;

while (i <= 5) {
    sum += i;
    i++;
}

console.log(sum); // 輸出: 15
```

### 使用 `while` 迴圈讀取陣列
以下範例顯示如何使用 `while` 迴圈遍歷一個陣列：

```typescript
const arr = [10, 20, 30, 40, 50];
let index = 0;

while (index < arr.length) {
    console.log(arr[index]);
    index++;
}
```

## 解釋
### 常見的陷阱
- **無限迴圈**：最常見的錯誤是未正確更新條件變數，導致條件永遠為真，造成程式無法結束。
- **條件評估**：確保條件的邏輯正確，避免因錯誤的條件導致不必要的錯誤或無法執行預期的程式碼。

### 附加資訊
- `while` 迴圈在執行效率上和其他迴圈（如 `for` 迴圈）相似，但在某些情況下可能使程式碼更具可讀性。
- 使用 `do...while` 迴圈可以確保至少執行一次程式碼，這是 `while` 迴圈所不具備的特性。

## 一句總結
`while` 迴圈是一種強大的控制結構，用於重複執行程式碼，直到特定條件為假。
