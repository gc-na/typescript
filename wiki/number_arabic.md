# الأعداد في TypeScript: فهم الأنواع العددية واستخداماتها

## ملخص
تعتبر الأعداد من الأنواع الأساسية في TypeScript، حيث توفر وسيلة للتعامل مع القيم الرقمية. يدعم TypeScript النوعين الأساسيين من الأعداد: الأعداد الصحيحة والأعداد العشرية، مما يسهل البرمجة الرياضية والتعامل مع البيانات العددية.

## الوثائق
في TypeScript، يُستخدم النوع `number` للتعبير عن القيم العددية. هذا النوع يشمل الأعداد الصحيحة (مثل 1، 2، 3) والأعداد العشرية (مثل 1.5، 2.75). يتم تخزين جميع الأعداد كنمط عددي مزدوج الدقة (double-precision floating-point) وفقًا لمعيار IEEE 754.

### الاستخدام
لإعلان متغير من النوع `number`، يمكن استخدام الصيغة التالية:
```typescript
let age: number = 30;
let price: number = 99.99;
```
يمكن إجراء العمليات الرياضية الأساسية مثل الجمع، الطرح، الضرب، والقسمة:
```typescript
let x: number = 10;
let y: number = 5;
let sum: number = x + y; // 15
let product: number = x * y; // 50
```

### التفاصيل
- **النطاق**: يدعم `number` نطاقًا واسعًا من القيم، من القيم السالبة الكبيرة إلى القيم الموجبة الكبيرة.
- **القيم الخاصة**: يمكن استخدام القيم الخاصة مثل `NaN` (ليس عددًا) و`Infinity` (للانفجار) و`-Infinity` (للانفجار السالب).
- **التقريب**: يمكنك استخدام دوال مثل `Math.round()`, `Math.floor()`, و`Math.ceil()` للتقريب.

## الأمثلة
### مثال 1: إعلان وتحليل الأعداد
```typescript
let length: number = 15;
let width: number = 20;
let area: number = length * width; // 300
```

### مثال 2: العمليات الرياضية
```typescript
let a: number = 12;
let b: number = 8;
let result: number = a - b; // 4
```

### مثال 3: التعامل مع القيم الخاصة
```typescript
let notANumber: number = NaN;
let infiniteValue: number = Infinity;
```

## الشرح
عند العمل مع نوع `number` في TypeScript، يجب أن تكون حذرًا من بعض القضايا الشائعة:
- **التقريب العشري**: عند التعامل مع الأعداد العشرية، قد تحدث أخطاء دقيقة بسبب الطريقة التي يتم بها تمثيلها في الذاكرة.
- **NaN**: يجب تجنب إجراء عمليات حسابية مع `NaN` لأنه سيؤدي دائمًا إلى نتيجة `NaN`.
- **المقارنة**: عند مقارنة الأعداد، تأكد من استخدام الدوال المناسبة، حيث أن `NaN` لا يساوي أي قيمة، بما في ذلك نفسه.

## ملخص بمقطع واحد
يعتبر `number` في TypeScript نوعًا أساسيًا لتمثيل الأعداد، مما يسهل التعامل مع القيم الرياضية والبيانات العددية.
