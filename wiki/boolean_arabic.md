# البوليان في تايب سكريبت: فهم القيم المنطقية

## الملخص
تعتبر القيم المنطقية (Boolean) في TypeScript نوعًا أساسيًا يُستخدم لتمثيل حالتين فقط: صحيح (true) أو خطأ (false). 

## الوثائق
### الغرض
يُستخدم نوع البيانات البولياني في TypeScript لتسهيل اتخاذ القرارات في البرمجة، مثل التحكم في تدفق البرنامج باستخدام العبارات الشرطية.

### الاستخدام
يمكن تعريف المتغيرات من نوع البوليان باستخدام الكلمة المفتاحية `boolean`. يتم تعيين قيم المتغيرات البوليانية إلى `true` أو `false` فقط. يُعتبر هذا النوع مفيدًا بشكل خاص في الحلقات، الشروط، والتقييمات المنطقية.

### التفاصيل
- **الإعلان عن المتغيرات**: يمكن الإعلان عن المتغيرات البوليانية باستخدام `let` أو `const`.
- **العمليات المنطقية**: يمكن استخدام العمليات المنطقية مثل `&&` (AND)، `||` (OR)، و `!` (NOT) لمعالجة القيم البوليانية.

## أمثلة
### مثال 1: تعريف متغير بولياني
```typescript
let isActive: boolean = true;
```

### مثال 2: استخدام الشرط
```typescript
let isUserLoggedIn: boolean = false;

if (isUserLoggedIn) {
    console.log("مرحبًا بك!");
} else {
    console.log("يرجى تسجيل الدخول.");
}
```

### مثال 3: العمليات المنطقية
```typescript
let hasPermission: boolean = true;
let isAdmin: boolean = false;

if (hasPermission && isAdmin) {
    console.log("يمكنك الوصول إلى هذه الصفحة.");
} else {
    console.log("ليس لديك إذن.");
}
```

## الشرح
### الأخطاء الشائعة
- **عدم التحقق من القيم**: يجب التأكد من أن المتغيرات البوليانية قد تم تعيين قيم صحيحة (true أو false) قبل استخدامها في الشروط.
- **استخدام القيم غير المنطقية**: تأكد من عدم تمرير قيم غير منطقية (مثل الأعداد أو السلاسل النصية) إلى المتغيرات البوليانية.

### ملاحظات إضافية
- يمكن استخدام القيم البوليانية في العمليات المنطقية لتبسيط الشروط.
- لا تنس أن TypeScript يوفر التحقق من النوع، مما يساعد في تجنب الأخطاء المرتبطة بنوع البيانات.

## ملخص عبارة واحدة
البوليان في TypeScript هو نوع بيانات يُستخدم لتمثيل القيم المنطقية فقط: صحيح أو خطأ.
