# TypeScript 中的 for 循环：用法与示例

## 概述
在 TypeScript 中，`for` 循环是一种用于重复执行代码块的控制结构。它的主要用途是遍历数组、对象或执行固定次数的操作。

## 文档
`for` 循环的基本语法如下：

```typescript
for (初始化; 条件; 更新) {
    // 循环体
}
```

### 目的
`for` 循环用于在满足特定条件时重复执行一段代码。它可以通过初始化表达式设置循环的初始状态，通过条件表达式控制循环的执行，通过更新表达式更新循环的状态。

### 用法
- **初始化**：在循环开始之前执行，只执行一次。
- **条件**：每次循环迭代之前检查，如果为真则执行循环体。
- **更新**：每次循环体执行完后执行，用于更新循环变量的值。

## 示例
### 示例 1：基本的 for 循环
```typescript
for (let i = 0; i < 5; i++) {
    console.log(i);
}
```
输出：
```
0
1
2
3
4
```

### 示例 2：遍历数组
```typescript
const fruits = ['苹果', '香蕉', '橘子'];
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}
```
输出：
```
苹果
香蕉
橘子
```

### 示例 3：双重 for 循环
```typescript
for (let i = 0; i < 3; i++) {
    for (let j = 0; j < 3; j++) {
        console.log(`i: ${i}, j: ${j}`);
    }
}
```
输出：
```
i: 0, j: 0
i: 0, j: 1
i: 0, j: 2
i: 1, j: 0
i: 1, j: 1
i: 1, j: 2
i: 2, j: 0
i: 2, j: 1
i: 2, j: 2
```

## 说明
- **常见错误**：在 `for` 循环中，忘记更新循环变量可能导致无限循环。
- **作用域**：在 `for` 循环中使用 `let` 声明变量时，变量的作用域仅限于循环内部。
- **性能考虑**：对于大型数组，使用 `for` 循环可能比使用其他数组迭代方法（如 `forEach`）更高效。

## 一句话总结
TypeScript 中的 `for` 循环是一种强大的控制结构，用于重复执行代码块并遍历数据。
