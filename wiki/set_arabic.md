# المجموعة (Set) في TypeScript: فهم شامل

## الملخص
المجموعة (Set) في TypeScript هي بنية بيانات تسمح بتخزين قيم فريدة من نوعها، مما يسهل إدارة البيانات والمساعدة في عمليات البحث والتحقق من التكرارات.

## الوثائق
تعتبر المجموعة (Set) في TypeScript جزءًا من ECMAScript 2015 (ES6) وتوفر وسيلة لتخزين عناصر فريدة، حيث يمكن استخدامها لتخزين أي نوع من القيم، بما في ذلك القيم الأولية والكائنات. يمكن استخدام المجموعة لتجنب القيم المكررة وتسهيل العمليات على البيانات.

### الغرض
تستخدم المجموعة لتوفير أداء أفضل عند الحاجة إلى التحقق من وجود عنصر أو لإجراء عمليات على البيانات المكررة.

### الاستخدام
لإنشاء مجموعة جديدة، يمكن استخدام الكود التالي:

```typescript
let mySet = new Set<number>();
```

لإضافة عناصر إلى المجموعة، يمكن استخدام طريقة `add`:

```typescript
mySet.add(1);
mySet.add(2);
mySet.add(3);
```

لإزالة عنصر، يمكنك استخدام طريقة `delete`:

```typescript
mySet.delete(2);
```

للتحقق من وجود عنصر معين، استخدم طريقة `has`:

```typescript
mySet.has(1); // true
```

للحصول على عدد العناصر في المجموعة، يمكنك استخدام خاصية `size`:

```typescript
console.log(mySet.size); // 2
```

## الأمثلة
### مثال 1: إنشاء مجموعة وإضافة عناصر

```typescript
let fruits = new Set<string>();
fruits.add("تفاح");
fruits.add("موز");
fruits.add("برتقال");
console.log(fruits.size); // 3
```

### مثال 2: التحقق من وجود عنصر

```typescript
console.log(fruits.has("موز")); // true
console.log(fruits.has("عنب")); // false
```

### مثال 3: إزالة عنصر

```typescript
fruits.delete("موز");
console.log(fruits.has("موز")); // false
```

## الشرح
### الأخطاء الشائعة
- **عدم فهم القيم الفريدة**: قد يعتقد البعض أن المجموعة تسمح بتكرار القيم، ولكنها مصممة فقط لتخزين القيم الفريدة.
- **التحويل التلقائي**: عند إضافة كائنات، يتم اعتبار كائنات مختلفة كقيم فريدة حتى لو كانت تحتوي على نفس الخصائص.

### ملاحظات إضافية
- يمكن استخدام المجموعة مع أي نوع من البيانات، بما في ذلك الكائنات المعقدة.
- الأداء في عمليات البحث والتحقق يكون عادة أفضل من استخدام مصفوفات أو هياكل بيانات أخرى.

## ملخص جملة واحدة
تعتبر المجموعة (Set) في TypeScript أداة فعالة لتخزين قيم فريدة وإدارة البيانات بشكل فعال.
