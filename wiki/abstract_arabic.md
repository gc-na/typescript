# الكلمة المحجوبة (abstract) في TypeScript: مفهومها واستخداماتها

## الملخص
تعتبر الكلمة المحجوبة "abstract" في TypeScript أداة هامة تستخدم لتعريف الفئات المجردة، والتي توفر هيكلًا أساسيًا للفئات الفرعية، مما يسهل بناء تطبيقات معقدة بطريقة منظمة وفعالة.

## الوثائق
تستخدم الكلمة المحجوبة "abstract" للإشارة إلى أن الفئة أو الطريقة غير مكتملة، مما يعني أنه لا يمكن إنشاء كائنات مباشرة من هذه الفئة. بدلاً من ذلك، يجب على الفئات الفرعية أن ترث من الفئة المجردة وتقوم بتطبيق الوظائف المعلنة.

### الغرض
الغرض من استخدام الكلمة المحجوبة "abstract" هو وضع إطار عمل لفئات معينة وتشجيع تطوير فئات فرعية تقوم بتطبيق هذه الوظائف.

### الاستخدام
يمكن استخدام الكلمة المحجوبة في تعريف الفئات والمناهج كما يلي:

1. **تعريف الفئة المجردة**:
   ```typescript
   abstract class Animal {
       abstract makeSound(): void;
   }
   ```

2. **تعريف فئة فرعية**:
   ```typescript
   class Dog extends Animal {
       makeSound(): void {
           console.log("Woof!");
       }
   }
   ```

## الأمثلة
### مثال 1: تعريف فئة مجرّدة
```typescript
abstract class Shape {
   abstract area(): number;
}

class Circle extends Shape {
   constructor(private radius: number) {
       super();
   }
   area(): number {
       return Math.PI * this.radius ** 2;
   }
}
```

### مثال 2: استخدام الفئات المجردة
```typescript
const myCircle = new Circle(5);
console.log(myCircle.area()); // Outputs: 78.53981633974483
```

## الشرح
### النقاط الشائعة
1. **عدم القدرة على إنشاء كائنات مباشرة**: لا يمكنك إنشاء كائن من الفئة المجردة مباشرة، يجب أن تعتمد على فئة فرعية.
2. **تنفيذ الطرق المجردة**: يجب على الفئات الفرعية تنفيذ جميع الطرق المجردة المعلن عنها في الفئة المجردة.

### ملاحظات إضافية
- الفئات المجردة تعزز من استخدام البرمجة الكائنية (OOP) وتساعد في تنظيم الكود.
- من المهم أن تتذكر أن الفئة المجردة يمكن أن تحتوي على طرق عادية، بالإضافة إلى الطرق المجردة.

## ملخص بجملة واحدة
تعتبر الكلمة المحجوبة "abstract" في TypeScript أداة فعالة لتعريف الفئات المجردة، مما يسهل بناء تطبيقات منظمة وقابلة للتطوير.
