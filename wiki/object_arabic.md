# كائن في TypeScript: دليل شامل للمطورين

## ملخص
يعد "الكائن" (Object) في TypeScript بنية بيانات أساسية تُستخدم لتخزين مجموعات من القيم والوظائف في شكل أزواج من المفتاح والقيمة. يوفر TypeScript ميزات إضافية تعزز من استخدام الكائنات، مثل الأنواع القابلة للتخصيص والتحقق من الأخطاء في وقت الترجمة.

## الوثائق
### الغرض
يهدف الكائن في TypeScript إلى تنظيم البيانات بطريقة منطقية ومهيكلة، مما يسهل التعامل مع البيانات المعقدة وتطوير التطبيقات الكبيرة.

### الاستخدام
يمكن تعريف الكائنات في TypeScript باستخدام الصيغة التالية:
```typescript
let myObject: { [key: string]: any } = {
    key1: value1,
    key2: value2,
    // المزيد من المفاتيح والقيم
};
```
يمكن أن تحتوي الكائنات على خصائص من أنواع مختلفة، بما في ذلك الأرقام، السلاسل، وحتى الدوال. يتميز TypeScript بتوفير القدرة على تحديد الأنواع بشكل صارم، مما يساعد في تجنب الأخطاء البرمجية.

### تفاصيل
- **تحديد الأنواع:** يمكن تحديد أنواع الخصائص داخل الكائن، مما يعزز من سلامة الكود. على سبيل المثال:
  ```typescript
  let user: { name: string; age: number; } = {
      name: "أحمد",
      age: 30
  };
  ```

- **الكائنات المتداخلة:** يمكن أن تحتوي الكائنات على كائنات أخرى، مما يتيح إنشاء هياكل بيانات معقدة:
  ```typescript
  let user = {
      name: "أحمد",
      address: {
          street: "123 الشارع الرئيسي",
          city: "الرياض"
      }
  };
  ```

- **الدوال:** يمكن أن تحتوي الكائنات على دوال، مما يسمح بتعريف سلوك الكائن:
  ```typescript
  let calculator = {
      add: function(a: number, b: number) {
          return a + b;
      }
  };
  ```

## الأمثلة
### مثال 1: كائن بسيط
```typescript
let car = {
    brand: "تويوتا",
    model: "كامري",
    year: 2022
};

console.log(car.brand); // تويوتا
```

### مثال 2: كائن مع دالة
```typescript
let person = {
    name: "علي",
    greet: function() {
        console.log(`مرحبا، أنا ${this.name}`);
    }
};

person.greet(); // مرحبا، أنا علي
```

### مثال 3: كائن متداخل
```typescript
let company = {
    name: "شركة التكنولوجيا",
    location: {
        city: "دبي",
        country: "الإمارات العربية المتحدة"
    }
};

console.log(company.location.city); // دبي
```

## الشرح
### الأخطاء الشائعة
- **نسيان تحديد الأنواع:** قد يؤدي عدم تحديد أنواع الخصائص إلى أخطاء في وقت التشغيل. يُفضل دائمًا استخدام الأنواع لضمان دقة البيانات.
- **الوصول إلى خصائص غير موجودة:** يمكن أن يحدث خطأ عند محاولة الوصول إلى خاصية غير معرفة، لذا يُفضل التحقق من وجود الخاصية أولاً.

### ملاحظات إضافية
- يمكن استخدام الكائنات كوسيلة لتعزيز تنظيم الكود من خلال إنشاء هياكل بيانات واضحة وسهلة الفهم.
- يوفر TypeScript دعمًا قويًا للعمل مع الكائنات، مما يجعل من السهل تطوير تطبيقات كبيرة ومعقدة.

## ملخص جملة واحدة
الكائن في TypeScript هو بنية بيانات رئيسية تستخدم لتخزين القيم والوظائف في شكل أزواج من المفتاح والقيمة، مما يسهل تنظيم البيانات وتطوير التطبيقات.
