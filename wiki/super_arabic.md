# استخدام الكلمة المفتاحية "super" في TypeScript

## ملخص
الكلمة المفتاحية "super" في TypeScript تُستخدم للوصول إلى خصائص وأعضاء الكائنات من الفئة الأساسية (الأب) في الفئات المشتقة (الأبناء). توفر هذه الكلمة المفتاحية وسيلة فعالة للتعامل مع الوراثة في TypeScript.

## الوثائق
تُعتبر "super" جزءًا أساسيًا من نظام الوراثة في TypeScript، حيث تُستخدم لتسهيل الوصول إلى الخصائص والأساليب الموجودة في الفئة الأساسية. عند تعريف فئة جديدة ترث من فئة أخرى، يمكنك استخدام "super" لاستدعاء المُنشئ (constructor) أو الوصول إلى الأساليب أو الخصائص المباشرة من الفئة الأساسية.

### الاستخدام
- **استدعاء المُنشئ**: يمكنك استخدام `super()` في المُنشئ لفئة مشتقة لاستدعاء المُنشئ الخاص بالفئة الأساسية.
- **الوصول إلى أساليب الفئة الأساسية**: يمكن استخدام `super.methodName()` لاستدعاء أساليب محددة من الفئة الأساسية.

### التفاصيل
- يُعتبر استخدام "super" ضروريًا عند الحاجة لاستدعاء المُنشئ للفئة الأساسية قبل تنفيذ أي منطق خاص بالفئة المشتقة.
- يجب أن تُستدعى "super()" قبل استخدام `this` داخل المُنشئ للفئة المشتقة.

## أمثلة
### مثال 1: استدعاء المُنشئ
```typescript
class Animal {
    constructor(public name: string) {
        console.log(`${name} is an animal.`);
    }
}

class Dog extends Animal {
    constructor(name: string) {
        super(name); // استدعاء المُنشئ الخاص بالفئة الأساسية
        console.log(`${name} is a dog.`);
    }
}

const dog = new Dog('Rex');
```

### مثال 2: الوصول إلى أساليب الفئة الأساسية
```typescript
class Animal {
    makeSound() {
        console.log('Animal sound');
    }
}

class Dog extends Animal {
    makeSound() {
        super.makeSound(); // استدعاء أسلوب الفئة الأساسية
        console.log('Bark');
    }
}

const dog = new Dog();
dog.makeSound();
```

## الشرح
- **المخاطر الشائعة**: قد يواجه المبرمجون الجدد مشكلات عند استخدام "super" إذا حاولوا استخدامها قبل استدعاء `super()` في المُنشئ. يجب أن تكون "super()" في أول سطر من المُنشئ للفئة المشتقة.
- **ملاحظات إضافية**: إذا كانت الفئة الأساسية تحتوي على مُنشئ بدون معلمات، يمكنك استدعاء `super()` بدون أي معلمات، ولكن إذا كان هناك مُنشئ يأخذ معلمات، يجب تمرير المعلمات المناسبة.

## ملخص من سطر واحد
تُستخدم الكلمة المفتاحية "super" في TypeScript للوصول إلى خصائص وأساليب الفئات الأساسية من الفئات المشتقة، مما يسهل إدارة الوراثة.
