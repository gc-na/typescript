# TypeScriptの「while」ループの使い方と特徴

## 概要
TypeScriptにおける「while」ループは、条件が真の間、特定のコードブロックを繰り返し実行するための制御フロー構文です。この文は、無限ループを作成したり、特定の条件が満たされるまで処理を続ける際に非常に便利です。

## ドキュメンテーション
### 目的
「while」ループは、特定の条件が真である限り、コードを繰り返し実行するために使用されます。これは、動的なデータ処理や、条件に基づいて反復処理を行う際に役立ちます。

### 使用法
基本的な構文は以下の通りです：

```typescript
while (条件) {
    // 繰り返し実行するコード
}
```

- **条件**: ブール値を返す式で、真であればループ内のコードが実行されます。
- ループ内で条件を変化させる場合は、必ず条件が偽になるように制御する必要があります。

### 詳細
- **初期化**: ループの前に変数を初期化します。
- **条件評価**: 「while」ループは、各反復で条件を評価します。
- **終了条件**: 条件が偽になると、ループが終了します。

## 例
以下に「while」ループの基本的な使用例を示します。

### 例1: シンプルなカウントアップ

```typescript
let count = 0;

while (count < 5) {
    console.log(count);
    count++;
}
```

この例では、`count`が5未満の間、`count`の値をコンソールに出力し、毎回1を加算します。

### 例2: ユーザー入力の繰り返し処理

```typescript
let input: string;
let validInputs: string[] = [];

while (input !== 'exit') {
    input = prompt('入力してください（終了するには exit と入力）');
    if (input && input !== 'exit') {
        validInputs.push(input);
    }
}

console.log(validInputs);
```

この例では、ユーザーが「exit」と入力するまで、入力を受け取り続けます。

## 説明
### 一般的な落とし穴
- **無限ループ**: 条件が常に真の場合、ループは終了しません。必ず終了条件を設定し、ループの内部で条件を変更するように注意が必要です。
- **パフォーマンス**: 複雑な条件を持つループや重い処理がループ内にある場合、アプリケーションのパフォーマンスに影響を与える可能性があります。

### 注意点
- 「while」ループの条件が変更されない場合、プログラムがフリーズすることがあります。これを避けるために、条件を更新するロジックを必ず実装しましょう。
- TypeScriptでは型安全が保証されるため、条件に使用する変数の型を明示することで、より安全なコードを書くことが可能です。

## 一文での要約
TypeScriptの「while」ループは、指定された条件が真である限り、コードブロックを繰り返し実行するための制御構文です。
