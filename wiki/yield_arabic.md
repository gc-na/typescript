# استخدام "yield" في TypeScript: مفهوم وأمثلة

## ملخص
تُستخدم كلمة "yield" في TypeScript للإشارة إلى نقطة تعود فيها وظيفة المولد (Generator Function) بنتائج مؤقتة. تُعتبر هذه الكلمة أساسية في كتابة شيفرات غير متزامنة وتسهيل إدارة التدفقات.

## الوثائق
تُستخدم كلمة "yield" في TypeScript داخل مولدات الوظائف، والتي تُعرف بأنها دوال تُستخدم لإنشاء كائنات يمكنها إرجاع قيم متعددة على مراحل. عند استدعاء دالة مولد، يتم تنفيذ الشيفرة حتى يتم الوصول إلى كلمة "yield"، عندها يتم إرجاع القيمة الموجودة بعد الكلمة، ويتم حفظ حالة الدالة. عند استئناف الدالة، تُستأنف من النقطة التي توقفت عندها.

### الهدف
الغرض من "yield" هو توفير طريقة مرنة للتعامل مع البيانات المتدفقة، مما يسمح بإنتاج القيم بشكل متتابع بدلاً من إرجاعها دفعة واحدة.

### الاستخدام
لاستخدام "yield"، يجب أن يتم تعريف الدالة باستخدام الكلمة المفتاحية `function*` (نجمة بعد كلمة "function")، مما يشير إلى أنها دالة مولد. يمكن استخدام "yield" داخل هذه الدالة لإرجاع قيمة ثم استئناف التنفيذ لاحقًا.

## أمثلة

### مثال 1: دالة مولد بسيطة
```typescript
function* generateNumbers() {
    yield 1;
    yield 2;
    yield 3;
}

const generator = generateNumbers();
console.log(generator.next().value); // 1
console.log(generator.next().value); // 2
console.log(generator.next().value); // 3
```

### مثال 2: استخدام "yield" في حلقة
```typescript
function* countDown(from: number) {
    while (from >= 0) {
        yield from;
        from--;
    }
}

const countdown = countDown(5);
for (const number of countdown) {
    console.log(number); // 5, 4, 3, 2, 1, 0
}
```

## الشرح
عند استخدام "yield"، يجب الانتباه إلى بعض النقاط:

1. **تأجيل التنفيذ**: كل مرة يتم فيها استدعاء `next()` على كائن المولد، يتم استئناف التنفيذ من آخر نقطة تم الوصول إليها.
2. **القيم المبدئية**: يمكن تمرير قيمة إلى `next()` والتي تكون متاحة كقيمة عائدة من الكلمة "yield" السابقة.
3. **الدوائر غير المنتهية**: يجب الحذر من كتابة مولدات قد تؤدي إلى دوائر غير منتهية إذا لم يتم السيطرة على تدفق القيم.

## ملخص جملة واحدة
تُستخدم "yield" في TypeScript لإنتاج قيم متعددة من دالة مولد، مما يسهل التعامل مع البيانات المتدفقة.
