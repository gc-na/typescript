# TypeScript의 "if" 문: 조건문 활용하기

## 개요
TypeScript에서 "if" 문은 조건에 따라 코드 블록의 실행 여부를 결정하는 데 사용되는 기본 제어 구조입니다. 이 문은 조건이 참일 때 특정 코드를 실행하고, 그렇지 않을 경우 다른 코드를 실행할 수 있도록 합니다.

## 문서화
### 목적
"if" 문은 주어진 조건의 참/거짓 여부에 따라 프로그램의 흐름을 제어하는 데 필수적입니다. TypeScript의 강력한 타입 시스템과 결합하여, 더욱 안전하고 예측 가능한 조건문을 작성할 수 있습니다.

### 사용법
TypeScript에서 "if" 문은 다음과 같은 기본 구조를 가지고 있습니다:

```typescript
if (조건) {
    // 조건이 참일 때 실행할 코드
} else {
    // 조건이 거짓일 때 실행할 코드 (선택적)
}
```

또한, 여러 조건을 검사할 필요가 있을 때는 `else if` 문을 활용할 수 있습니다.

```typescript
if (조건1) {
    // 조건1이 참일 때 실행할 코드
} else if (조건2) {
    // 조건2가 참일 때 실행할 코드
} else {
    // 위 조건들이 모두 거짓일 때 실행할 코드
}
```

## 예제
1. 기본 `if` 문 사용 예제:
```typescript
let age: number = 20;

if (age >= 18) {
    console.log("성인입니다.");
} else {
    console.log("미성년자입니다.");
}
```

2. `else if` 문 사용 예제:
```typescript
let score: number = 85;

if (score >= 90) {
    console.log("A");
} else if (score >= 80) {
    console.log("B");
} else if (score >= 70) {
    console.log("C");
} else {
    console.log("F");
}
```

## 설명
- **조건 표현식**: "if" 문에서 사용되는 조건은 불리언 값으로 평가될 수 있는 표현식이어야 합니다. 예를 들어, 숫자 비교, 문자열 비교 등을 포함할 수 있습니다.
- **참조 타입**: TypeScript는 정적 타입을 지원하므로, 조건문에서 사용하는 변수의 타입을 명확하게 정의하는 것이 좋습니다. 이는 코드의 가독성을 높이고, 타입 관련 오류를 줄이는 데 도움을 줍니다.

### 일반적인 함정
- **부주의한 조건**: 조건문에서 잘못된 비교를 사용하면 의도하지 않은 결과를 초래할 수 있습니다. 예를 들어, `=` (대입 연산자) 대신 `==` 또는 `===` (비교 연산자)를 사용하는 것을 잊지 않도록 주의해야 합니다.
- **조건 평가**: 다중 조건을 평가할 때, 순서가 중요합니다. 먼저 평가할 조건을 잘 고려하여 작성해야 합니다.

## 한 줄 요약
TypeScript의 "if" 문은 조건에 따라 코드 실행을 제어하는 필수적인 제어 구조입니다.
