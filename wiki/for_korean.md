# TypeScript에서의 for 반복문: 사용법과 예제

## 개요
TypeScript에서 `for` 반복문은 특정 조건을 만족할 때까지 코드 블록을 반복 실행하는 구조를 제공합니다. JavaScript의 `for` 문과 동일하게 작동하며, 효율적인 루프를 통해 배열이나 객체의 요소를 순회할 수 있습니다.

## 문서화
`for` 문은 TypeScript 및 JavaScript에서 가장 일반적인 반복문 중 하나입니다. 이 문법을 사용하면 초기화, 조건 검사, 후속 작업을 통해 반복을 제어할 수 있습니다. 일반적인 형태는 다음과 같습니다:

```typescript
for (초기화; 조건; 후속 작업) {
    // 반복 실행할 코드
}
```

- **초기화**: 반복을 시작하기 위한 변수를 초기화합니다.
- **조건**: 반복을 계속할지를 결정하는 불리언 표현식입니다. 이 값이 `true`인 경우 코드 블록이 실행됩니다.
- **후속 작업**: 각 반복이 끝난 후 실행되는 코드입니다. 보통 카운터 변수를 증가시키는 데 사용됩니다.

이 구조는 배열, 문자열, 객체 등 다양한 데이터 구조를 처리하는 데 유용합니다.

## 예제
### 기본 사용 예제
```typescript
// 0부터 4까지의 숫자를 출력
for (let i = 0; i < 5; i++) {
    console.log(i);
}
```

### 배열 순회 예제
```typescript
const fruits: string[] = ['사과', '바나나', '체리'];

for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}
```

### 객체의 속성 순회 예제
```typescript
const person = {
    name: '홍길동',
    age: 30,
    job: '개발자'
};

for (const key in person) {
    console.log(`${key}: ${person[key]}`);
}
```

## 설명
`for` 반복문을 사용할 때 주의해야 할 몇 가지 사항이 있습니다:

1. **무한 루프**: 조건이 항상 `true`인 경우, 반복문이 무한히 실행될 수 있습니다. 초기화 및 후속 작업을 적절히 설정해야 합니다.
2. **배열 인덱스 초과**: 배열의 길이를 초과하는 인덱스를 참조하면 `undefined`를 반환합니다. 항상 조건문에서 배열의 길이를 체크해야 합니다.
3. **변수 스코프**: `let` 키워드를 사용하여 선언한 변수는 블록 스코프를 가지므로, 루프 외부에서 해당 변수를 참조할 수 없습니다. 반면, `var`는 함수 스코프를 가집니다.

## 한줄 요약
TypeScript의 `for` 반복문은 초기화, 조건 검사, 후속 작업을 통해 코드 블록을 반복 실행하는 구조로, 배열 및 객체의 요소를 순회하는 데 유용합니다.
