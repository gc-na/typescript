# مساحة الأسماء في TypeScript: دليل شامل

## ملخص
مساحة الأسماء في TypeScript هي ميزة تُستخدم لتنظيم الكود وتقليل تداخل الأسماء، مما يُساعد المطورين في إدارة الشيفرة بشكل أكثر فعالية.

## الوثائق
تُعتبر مساحة الأسماء وسيلة لتجميع المتغيرات، الدوال، والواجهات تحت اسم واحد، مما يُتيح لك استخدام الأسماء بدون الحاجة إلى تكرارها. تُستخدم المساحات بشكل رئيسي في المشاريع الكبيرة لتفادي الصراعات بين الأسماء.

### الغرض
الغرض من مساحة الأسماء هو توفير هيكل منظم للكود، مما يسهل قراءة الشيفرة وفهمها، بالإضافة إلى تقليل احتمالية حدوث أخطاء ناتجة عن تداخل الأسماء.

### الاستخدام
يمكن تعريف مساحة الأسماء باستخدام الكلمة المفتاحية `namespace`، متبوعة باسم المساحة. يمكن بعد ذلك إضافة متغيرات، دوال، أو واجهات داخل هذه المساحة. 

```typescript
namespace MyNamespace {
    export function greet(name: string): string {
        return `Hello, ${name}!`;
    }
}
```

### تفاصيل
- يجب استخدام الكلمة المفتاحية `export` لجعل العناصر داخل مساحة الأسماء متاحة للاستخدام من خارجها.
- يمكن استيراد مساحة الأسماء في ملفات أخرى باستخدام الكلمة المفتاحية `import`.

## أمثلة
### مثال أساسي
```typescript
namespace Geometry {
    export function areaOfCircle(radius: number): number {
        return Math.PI * radius * radius;
    }
}

console.log(Geometry.areaOfCircle(5)); // الناتج: 78.53981633974483
```

### مثال مع استخدام دوال متعددة
```typescript
namespace MathOperations {
    export function add(a: number, b: number): number {
        return a + b;
    }

    export function subtract(a: number, b: number): number {
        return a - b;
    }
}

console.log(MathOperations.add(10, 5)); // الناتج: 15
console.log(MathOperations.subtract(10, 5)); // الناتج: 5
```

## الشرح
### الفخاخ الشائعة
- **عدم استخدام `export`:** إذا لم يتم تصدير العناصر، فلن تكون متاحة للاستخدام خارج مساحة الأسماء.
- **تداخل الأسماء:** رغم أن المساحات تُساعد في تجنب تداخل الأسماء، إلا أنه قد يحدث تداخل إذا تم استخدام نفس اسم المساحة في أماكن مختلفة.

### ملاحظات إضافية
- يُفضل استخدام المساحات في المشاريع الكبيرة، حيث أن الشيفرة تكون أكثر تعقيدًا.
- يمكن استخدام المساحات مع الوحدات (modules) في TypeScript، مما يوفر مزيدًا من التنظيم.

## ملخص جملة واحدة
مساحة الأسماء في TypeScript تُعتبر أداة فعالة لتنظيم الكود وتقليل تداخل الأسماء في المشاريع الكبيرة.
