# الكلمة المفتاحية "return" في TypeScript: كيفية استخدامها وأهميتها

## الملخص
تُستخدم الكلمة المفتاحية "return" في TypeScript للإشارة إلى انتهاء دالة معينة وإرجاع قيمة منها. تعتبر هذه الكلمة أساسية في البرمجة، حيث تحدد ما يتم إرجاعه عند استدعاء الدالة.

## الوثائق
تُستخدم "return" في TypeScript داخل الدوال لتحديد القيمة التي سيتم إرجاعها عند انتهاء تنفيذ الدالة. عند استخدام الكلمة "return"، يتم إنهاء تنفيذ الدالة بشكل فوري، ويتم إرسال القيمة المحددة إلى المكان الذي تم استدعاء الدالة منه.

### الاستخدام
- تُستخدم "return" في أي دالة تُعرّف في TypeScript.
- يمكن أن تُرجع الدالة قيمة من أي نوع، مثل الأرقام، السلاسل النصية، الكائنات، أو حتى الدوال الأخرى.
- إذا لم تُحدد قيمة بعد "return"، ستُرجع الدالة `undefined` بشكل افتراضي.

### التفاصيل
- يجب أن تكون الجملة "return" هي الجملة الأخيرة في الدالة قبل إغلاقها.
- يمكن استخدام "return" في الدوال العادية، الدوال السهمية، والدوال غير المتزامنة.

## الأمثلة
### مثال 1: دالة بسيطة تُرجع قيمة عددية
```typescript
function add(a: number, b: number): number {
    return a + b;
}

const result = add(5, 3);
console.log(result); // 8
```

### مثال 2: دالة تُرجع قيمة نصية
```typescript
function greet(name: string): string {
    return `Hello, ${name}!`;
}

console.log(greet("Ali")); // Hello, Ali!
```

### مثال 3: دالة تُرجع كائنًا
```typescript
function createPerson(name: string, age: number) {
    return { name: name, age: age };
}

const person = createPerson("Sara", 25);
console.log(person); // { name: "Sara", age: 25 }
```

## الشرح
### الفخاخ الشائعة
- **نسيان استخدام "return"**: في حال لم يتم استخدام "return" في دالة مصممة لإرجاع قيمة، سيؤدي ذلك إلى إرجاع `undefined` مما قد يسبب أخطاء في البرنامج.
- **تعدد الجمل "return"**: يمكن استخدام عدة جمل "return" في دالة واحدة، لكن يجب الانتباه إلى أن الدالة ستنتهي عند أول جملة "return" تُنفذ.
- **الإرجاع من الدوال غير المتزامنة**: عند استخدام "async" مع الدوال، يتم إرجاع قيمة كائن "Promise"، لذا يجب استخدام "await" للحصول على القيمة الحقيقية.

## جملة ملخصة
تُستخدم الكلمة المفتاحية "return" في TypeScript لإنهاء الدالة وإرجاع قيمة معينة إلى نقطة الاستدعاء.
