# استخدام "instanceof" في TypeScript: دليل شامل

## ملخص
يعتبر "instanceof" أداة قوية في TypeScript تُستخدم للتحقق مما إذا كان كائن معين هو مثيل لفئة معينة أو نوع محدد. يساعد هذا في ضمان سلامة الأنواع أثناء تنفيذ البرمجيات، مما يُعزز من موثوقية الكود.

## الوثائق
### الغرض
تتيح لك "instanceof" تحديد ما إذا كان كائن ما ينتمي إلى نوع معين أو فئة معينة، مما يُساعد في تنفيذ عمليات مختلفة بناءً على نوع الكائن.

### الاستخدام
تُستخدم "instanceof" على النحو التالي:
```typescript
object instanceof ClassName
```
حيث:
- `object`: الكائن الذي تريد التحقق منه.
- `ClassName`: اسم الفئة أو النوع الذي ترغب في التحقق مما إذا كان الكائن مثيلاً له.

### التفاصيل
1. **التحقق من الأنواع**: تُستخدم "instanceof" بشكل رئيسي للتحقق من الأنواع في هياكل البرمجة القائمة على الفئات.
2. **الوراثة**: تأخذ "instanceof" بعين الاعتبار تسلسل الوراثة، مما يعني أنه إذا كانت الفئة A هي فئة فرعية من الفئة B، فإن كائنًا من الفئة A سيكون أيضًا مثيلًا للفئة B.
3. **التحقق من الأنواع في الأنواع المخصصة**: يمكن استخدام "instanceof" مع الأنواع المخصصة أو الكائنات التي تم إنشاؤها بواسطة الدوال.

## الأمثلة
### مثال 1: استخدام "instanceof" مع الفئات
```typescript
class Animal {}
class Dog extends Animal {}

let myDog = new Dog();
console.log(myDog instanceof Dog); // true
console.log(myDog instanceof Animal); // true
```

### مثال 2: استخدام "instanceof" مع الأنواع المخصصة
```typescript
function isString(value: any): value is string {
    return typeof value === 'string';
}

let value: any = "Hello";
if (isString(value)) {
    console.log(value.toUpperCase()); // "HELLO"
}
```

## الشرح
### الأخطاء الشائعة
1. **استخدام "instanceof" مع الأنواع غير المدعومة**: لا يمكن استخدام "instanceof" مع الأنواع البدائية مثل `number` أو `string`، حيث ستظهر النتيجة كـ `false`.
2. **تداخل الفئات**: عند استخدام "instanceof" مع الفئات المعقدة، تأكد من عدم الخلط بين الفئات أو الأنواع، حيث قد يؤدي ذلك إلى نتائج غير صحيحة.

### ملاحظات إضافية
- يُفضل استخدام "instanceof" مع الكائنات التي تم إنشاؤها بواسطة الفئات بدلاً من الأنواع البدائية.
- تذكر أن "instanceof" يعتمد على السياق، لذا تأكد من أن الكائنات والفئات في نفس السياق.

## ملخص جملة واحدة
تُستخدم "instanceof" في TypeScript للتحقق مما إذا كان كائن معين هو مثيل لفئة معينة أو نوع محدد، مما يُعزز من سلامة الأنواع في البرمجة.
