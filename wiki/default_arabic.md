# استخدام الكلمة المحجوزة "default" في TypeScript

## الملخص
تُستخدم الكلمة المحجوزة "default" في TypeScript للإشارة إلى قيمة افتراضية يتم تصديرها من وحدة ما، مما يسهل استيرادها في وحدات أخرى.

## الوثائق
الكلمة المحجوزة "default" تُستخدم بشكل أساسي في عمليات التصدير والاستيراد في TypeScript. عند استخدام "export default"، يمكنك تصدير قيمة واحدة من وحدة، مما يعني أنه يمكن استيرادها بدون الحاجة لتسمية معينة. هذه الميزة تعزز من تنظيم الشيفرة وتجعلها أكثر وضوحًا وسهولة في الاستخدام.

### الغرض
الغرض من استخدام "default" هو تبسيط عملية التصدير والاستيراد، مما يسمح للمطورين بتحديد قيمة أو كائن يتم استخدامه بشكل شائع دون الحاجة إلى تسميات معقدة.

### الاستخدام
تُستخدم الكلمة "default" في نوعين من العمليات:
1. **تصدير القيم**: عند تصدير قيمة واحدة من وحدة، يمكن استخدام "export default" للإشارة إلى أن هذه القيمة هي القيمة الافتراضية.
2. **استيراد القيم**: عند استيراد القيمة الافتراضية من وحدة، لا تحتاج إلى استخدام الأقواس.

## الأمثلة
### مثال على تصدير قيمة افتراضية
```typescript
// ملف: myModule.ts
const myFunction = () => {
    console.log("Hello from myFunction!");
};

export default myFunction;
```

### مثال على استيراد قيمة افتراضية
```typescript
// ملف: main.ts
import myFunction from './myModule';

myFunction(); // يطبع: Hello from myFunction!
```

### مثال على تصدير كائن كقيمة افتراضية
```typescript
// ملف: config.ts
const config = {
    apiUrl: "https://api.example.com",
    timeout: 5000
};

export default config;
```

### مثال على استيراد كائن كقيمة افتراضية
```typescript
// ملف: app.ts
import config from './config';

console.log(config.apiUrl); // يطبع: https://api.example.com
```

## الشرح
من الأمور الشائعة التي يجب الانتباه إليها عند استخدام "default":
- يمكنك تصدير قيمة واحدة فقط كقيمة افتراضية من الوحدة. إذا حاولت تصدير أكثر من قيمة افتراضية، سيحدث خطأ.
- عند استيراد قيمة افتراضية، لا تحتاج لاستخدام الأقواس مثل `{ }`، بل يمكنك استيرادها مباشرة باسمها.
- يمكن دمج "default" مع عمليات التصدير الأخرى، مثل عمليات التصدير المسماة، لكن يجب تمييزها بوضوح.

## ملخص جملة واحدة
تُستخدم الكلمة المحجوزة "default" في TypeScript لتسهيل تصدير واستيراد القيم الافتراضية من الوحدات.
