# الرموز في TypeScript: دليل شامل

## الملخص
الرموز (Symbols) في TypeScript هي نوع فريد من القيم تُستخدم لتعريف خصائص الكائنات بطريقة تضمن عدم تداخلها مع خصائص أخرى.

## الوثائق
### الغرض
تُعتبر الرموز نوعًا مميزًا يُستخدم في TypeScript وJavaScript لإنشاء معرّفات فريدة. كل رمز يتم إنشاؤه هو قيمة فريدة، مما يجعلها مثالية لإنشاء خصائص خاصة داخل الكائنات.

### الاستخدام
يمكن إنشاء رمز باستخدام الدالة `Symbol()`. يمكن استخدام الرموز كمعرّفات خصائص لكائنات مختلفة، مما يساعد في تجنب التصادم مع الخصائص الأخرى.

### التفاصيل
- **إنشاء رمز:** لإنشاء رمز، استخدم `Symbol()`:
  ```typescript
  const mySymbol = Symbol();
  ```
  
- **تسميات الرموز:** يمكنك إضافة وصف إلى الرمز عند إنشائه، مما يسهل تتبعه، لكن هذا الوصف ليس جزءًا من القيمة الفريدة:
  ```typescript
  const mySymbolWithDescription = Symbol('myDescription');
  ```

- **استخدام الرموز كخصائص:** يمكنك استخدام الرموز كخصائص في الكائنات:
  ```typescript
  const myObject = {
    [mySymbol]: 'value'
  };
  ```

## الأمثلة
### مثال 1: إنشاء واستخدام رموز
```typescript
const uniqueId = Symbol('id');

const user = {
  name: 'Ali',
  [uniqueId]: 12345
};

console.log(user[uniqueId]); // 12345
```

### مثال 2: استخدام الرموز مع القيم الافتراضية
```typescript
const secretKey = Symbol('secret');

const data = {
  name: 'Ahmed',
  [secretKey]: 'hiddenValue'
};

console.log(data[secretKey]); // hiddenValue
```

## الشرح
### الأخطاء الشائعة
- **عدم فهم الفريدة:** يُعتبر كل رمز فريد، لذا لا تتوقع أن يكون رمزان بنفس الوصف متساويين.
- **عدم الحاجة للرموز:** الكثير من المبرمجين يبالغون في استخدام الرموز في سيناريوهات بسيطة حيث يمكن استخدام الخصائص العادية.

### ملاحظات إضافية
- الرموز غير قابلة للتحويل إلى سلسلة نصية، مما يجعلها مثالية للاستخدام كمعرفات خاصة.
- لا تظهر الرموز في الحلقات مثل `for...in`، مما يساعد على الحفاظ على الخصوصية.

## ملخص بجملة واحدة
الرموز في TypeScript تُستخدم لإنشاء خصائص فريدة وآمنة داخل الكائنات، مما يمنع التصادم مع الخصائص الأخرى.
