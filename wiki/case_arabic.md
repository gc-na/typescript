# حالة (Case) في TypeScript: فهم واستخدام عبارة الحالة

## الملخص
عبارة الحالة (case) في TypeScript هي جزء من هيكل التحكم في التدفق، تستخدم لتسهيل تنفيذ التعليمات بناءً على القيم المختلفة.

## الوثائق
عبارة الحالة تستخدم لتحديد تنفيذ مختلف الكود بناءً على قيمة معينة. يتم استخدام عبارة `switch` في TypeScript لتحديد مجموعة من القيم المحتملة لمتغير معين، مما يسهل قراءة وصيانة التعليمات البرمجية. 

### الاستخدام
تستخدم عبارة `switch` كالتالي:

```typescript
switch (expression) {
    case value1:
        // تنفيذ الكود إذا كانت القيمة تساوي value1
        break;
    case value2:
        // تنفيذ الكود إذا كانت القيمة تساوي value2
        break;
    default:
        // تنفيذ الكود إذا لم تتطابق أي من القيم
}
```

### التفاصيل
- **التعبير**: هو القيمة التي سيتم فحصها.
- **الحالات (cases)**: تمثل القيم المحتملة التي يمكن أن تتطابق مع التعبير.
- **المقطع `break`**: يمنع تنفيذ الكود بعد الحالة المطابقة. إذا تم حذف `break`، سيستمر التنفيذ في الحالات التالية.
- **المقطع `default`**: اختيارى، يُنفذ إذا لم تتطابق أي من الحالات.

## الأمثلة
### مثال بسيط:
```typescript
let day: number = 2;
let dayName: string;

switch (day) {
    case 1:
        dayName = "الأحد";
        break;
    case 2:
        dayName = "الإثنين";
        break;
    case 3:
        dayName = "الثلاثاء";
        break;
    default:
        dayName = "غير معروف";
}

console.log(dayName); // الإثنين
```

### مثال مع عدم استخدام `break`:
```typescript
let fruit: string = "تفاح";
let color: string;

switch (fruit) {
    case "موز":
        color = "أصفر";
    case "تفاح":
        color = "أخضر";
    case "برتقال":
        color = "برتقالي";
    default:
        color = "غير معروف";
}

console.log(color); // أخضر
```

## الشرح
- **المزالق الشائعة**: من الشائع أن ينسى المبرمجون إضافة عبارة `break`، مما يؤدي إلى تنفيذ حالات إضافية غير مقصودة.
- **التحقق من القيم**: يجب أن تكون القيم في الحالات متوافقة مع نوع البيانات للتعبير، مثل الأرقام أو السلاسل النصية.
- **استخدام القيم المتعددة**: يمكن استخدام عدة قيم في حالة واحدة للفحص، مثل:
  ```typescript
  switch (fruit) {
      case "موز":
      case "تفاح":
          color = "فاكهة";
          break;
  }
  ```

## ملخص من جملة واحدة
عبارة الحالة في TypeScript تستخدم لتحديد تنفيذ الكود بناءً على قيم مختلفة باستخدام هيكل `switch`.
