# استخدام جملة "if" في TypeScript: دليل شامل

## ملخص
تعتبر جملة "if" واحدة من أبرز الجمل الشرطية في TypeScript، حيث تُستخدم لتحديد المسار الذي سيتبعه البرنامج بناءً على تقييم شرط معين. تتيح للمطورين اتخاذ قرارات منطقية ضمن الكود.

## الوثائق
جملة "if" في TypeScript تُستخدم لتنفيذ كود معين إذا تحقق شرط محدد. إذا كان الشرط صحيحًا (true)، يتم تنفيذ الكود الموجود داخل كتلة "if". بينما إذا كان الشرط غير صحيح (false)، يمكن استخدام جملة "else" لتحديد كود بديل.

### التركيب:
```typescript
if (condition) {
    // الكود الذي سيتم تنفيذه إذا تحقق الشرط
} else {
    // الكود البديل الذي سيتم تنفيذه إذا لم يتحقق الشرط
}
```

### الاستخدام:
- **الشرط (condition)**: يمكن أن يكون أي تعبير يُرجع قيمة منطقية (true أو false).
- **كتلة الكود**: يمكن أن تحتوي على مجموعة من الأوامر التي سيتم تنفيذها إذا كان الشرط صحيحًا.

## الأمثلة

### مثال أساسي:
```typescript
let age: number = 18;

if (age >= 18) {
    console.log("أنت بالغ.");
} else {
    console.log("أنت قاصر.");
}
```

### استخدام "else if":
```typescript
let score: number = 75;

if (score >= 90) {
    console.log("ممتاز");
} else if (score >= 75) {
    console.log("جيد جداً");
} else {
    console.log("يجب تحسين الأداء");
}
```

## الشرح
### المشكلات الشائعة:
- **عدم استخدام الأقواس**: في حالة عدم استخدام أقواس لتغليف كتلة الكود، سيؤدي ذلك إلى تنفيذ السطر التالي فقط إذا تحقق الشرط، مما قد يسبب أخطاء غير متوقعة.
  
```typescript
if (true)
    console.log("هذا سيتنفذ");
    console.log("هذا لن يتنفذ إذا كان الشرط صحيحًا");
```

### ملاحظات إضافية:
- يمكن استخدام جملة "if" بشكل متداخل (nested) لإنشاء شروط أكثر تعقيدًا، ولكن يجب الحذر لتفادي التعقيد المفرط.
- جملة "if" تدعم التعبيرات المنطقية، مما يسمح لك بدمج عدة شروط معًا.

## ملخص بنقطة واحدة
تُستخدم جملة "if" في TypeScript لاتخاذ قرارات شرطية وتنفيذ كود معين بناءً على تقييم شروط معينة.
